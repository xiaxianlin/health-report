# 体检报告饮食建议 - 问卷收集模板

> 用途：用户下单后收集基础信息、体检资料、饮食情况与目标
> 适用服务：体检报告饮食建议｜1v1定制一周配餐
> 交付口径：PDF 报告交付，无在线可视化报告，无文字答疑

---

## 一、发送给用户的引导文案

### 版本 A：简洁版
```
您好～为了帮您整理更适合的饮食建议，请尽量完整填写以下信息。
如果有体检报告、化验单，也可以一起发我，这样会更方便做得更有针对性。
```

### 版本 B：更完整版
```
您好～接下来我会根据您的基础信息、体检数据和饮食情况，整理一份饮食建议 PDF 报告。
为了让建议更贴合您的情况，请尽量完整填写下面的问卷内容；如果有近期体检报告或化验单，也建议一并发送。
```

## 四、适合做表单的字段版本

### 必填项
- 称呼
- 性别
- 年龄
- 身高（cm）
- 体重（kg）
- 目前最想改善的问题
- 平时饮食概况
- 是否有已知疾病 / 用药 / 医生提醒
- 是否方便提供体检报告

### 选填项
- 最近一次体检时间
- 主要异常指标
- 运动习惯
- 作息情况
- 忌口 / 过敏 / 不吃的食物
- 特别希望获得的建议类型

### 上传项
- 体检报告
- 化验单
- 其他补充资料

## 五、收集时的提醒话术

### 1. 用户信息填得太少
```
如果方便的话，可以再补充一下平时怎么吃、最近最想改善什么问题，还有是否有体检报告。
这些信息越完整，给您的建议会越更有针对性。
```

### 2. 用户没有体检报告
```
没有体检报告也可以做。
您把基础信息、饮食情况和目标尽量说清楚就行；如果后面有报告，也可以补发。
```

### 3. 用户只发了一句“帮我做饮食方案”
```
可以的～我先把问卷发您，您按顺序填一下。
有体检报告的话也可以一并发我，这样会更方便做得更细。
```

---

## 六、内部使用建议

- **优先收完整基础信息**：年龄、性别、身高、体重、目标
- **有体检报告时优先参考报告**，没有时按用户主诉和饮食情况做建议
- **如涉及明确疾病或治疗中情况**，交付时要再次提醒“仅作饮食参考，优先遵医嘱”
- **不要承诺医疗效果**，对外统一为饮食建议服务
