# 营养评估 Skill

基于健康档案计算个性化每日营养目标，生成营养评估报告。

## 输入

- `$ARGUMENTS`：健康档案文件路径
- 留空则读取 `results/<用户名>/健康档案.md`

## 处理步骤

### 步骤 1: 读取健康档案

提取：基本信息、异常指标、诊断疾病、用药（含营养交互）、过敏原

### 步骤 2: 读取参考资料

读取知识库：`knowledge/disease-diet-rules.md`、`knowledge/nutrient-reference.md`
运行脚本查看计算公式：用 Bash 执行 `node -e "console.log(require('./scripts/calculate-nutrition.js'))"` 获取 BMR/TDEE/宏量营养素计算方法

### 步骤 3: 检索公开营养方案参考（必须执行）

至少 **3 个 WebSearch 查询**（权威指南、疾病膳食方案、多病叠加策略），保存到 `knowledge/references/参考_营养评估_[关键词]_YYYY-MM-DD.md` 并验证文件已保存。

优先来源：中华医学会/中国营养学会指南、三甲医院营养科方案、丁香医生等权威平台

### 步骤 4: 计算基础能量需求

**BMR** — Mifflin-St Jeor 公式：
- 男性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 + 5
- 女性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 - 161

**TDEE** = BMR × 活动系数（久坐 1.2 / 轻度 1.375 / 中度 1.55 / 高度 1.725）

**目标能量**：根据体重目标调整（减重 -300~500 / 维持 / 增重 +200~300 kcal）

**宏量营养素**：根据疾病调整比例（蛋白质 XX% / 碳水 XX% / 脂肪 XX%），用 `calcMacroGrams()` 转换为克数

**膳食纤维**：目标 14g/1000kcal

用 Bash 验证计算结果：
```bash
node -e "
const { calcBMR, calcTDEE, calcTargetEnergy, calcMacroGrams } = require('./scripts/calculate-nutrition');
const bmr = calcBMR('男', 87.1, 170.5, 45);
const tdee = calcTDEE(bmr, '久坐');
console.log('BMR:', bmr, 'TDEE:', tdee);
console.log('Target:', calcTargetEnergy(tdee, '减重'));
console.log('Macros:', calcMacroGrams(1620, 22, 53, 25));
"
```

### 步骤 5: 疾病调整

根据诊断逐病查阅 `disease-diet-rules.md` 和检索到的参考方案调整营养目标。

**多疾病调整优先级**：肾病（蛋白最严）> 痛风（嘌呤）> 糖尿病（碳水）> 高血压（钠）> 高脂血症（脂肪）> 肝病 > 贫血 > 食物过敏

**叠加原则**：蛋白质/碳水/脂肪取最严值，微量营养素限制取最严补充取最高，禁忌食物合并

### 步骤 6: 药物-营养交互

检查用药清单，常见交互：
- 二甲双胍 → 监测/补充维生素 B12
- 利尿剂（噻嗪类）→ 注意钾监测
- 华法林 → 保持维生素 K 摄入稳定
- 他汀类 → 避免西柚/葡萄柚
- 质子泵抑制剂 → 关注钙、镁、B12 吸收

### 步骤 7: 生成营养评估报告

格式要求（完整内容，含计算过程）：

- **基础代谢计算**: BMR公式和值、活动水平及系数、TDEE计算、目标能量及体重管理目标
- **每日营养目标**: 宏量营养素表（能量、蛋白质、碳水、脂肪、纤维—目标值/占比/范围/来源），微量营养素重点（钠、钾、钙、铁等—RNI/治疗目标/原因/是否需补充剂）
- **饮食原则**: 根据疾病逐条列出
- **推荐食物清单**: 按类别列出
- **禁忌/限制食物表**: 食物/原因/严重程度（完全禁止|严格限制|适量控制）
- **药物-营养交互提示**
- **三餐能量分配表**: 早餐/午餐/晚餐/加餐的能量和百分比
- **参考来源**: 列出所有参考的公开方案

### 步骤 8: 保存报告

保存到 `results/<用户名>/营养评估.md`。完成后自动执行 `/data-transform <用户名>`。

## 注意事项

- 能量缺口不低于 1200 kcal/天（女）或 1500 kcal/天（男）
- 糖尿病患者定时定量三餐，不建议少食多餐（除非使用胰岛素且医嘱要求加餐）
- 检索结果仅作参考，不可替代临床营养师的专业判断
- 每个输出文件末尾必须包含免责声明：本方案由 AI 辅助生成，仅提供营养参考信息，不构成医疗诊断或治疗建议
