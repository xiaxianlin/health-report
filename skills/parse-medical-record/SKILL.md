# Parse Medical Record

从用户提供的病历资料中提取对营养方案制定有关键价值的健康信息。

## 输入来源（按优先级）

1. **体检报告**：PDF / 图片
2. **病历档案**：医院门诊/住院病历
3. **手动录入**：用户直接输入的文字描述

## 提取字段

### 基本信息
- 姓名（脱敏为 X先生/女士）
- 性别、年龄
- 身高(cm)、体重(kg)、BMI（自动计算）
- 腰围(cm)（如有）
- 血压

### 血液检查指标
| 指标 | 单位 | 参考范围 |
|---|---|---|
| 空腹血糖 | mmol/L | 3.9-6.1 |
| 糖化血红蛋白(HbA1c) | % | 4.0-6.0 |
| 总胆固醇 | mmol/L | <5.2 |
| LDL-C | mmol/L | <3.4 |
| HDL-C | mmol/L | >1.0 |
| 甘油三酯 | mmol/L | <1.7 |
| ALT | U/L | 男9-50 / 女7-35 |
| AST | U/L | 男15-40 / 女13-35 |
| 白蛋白 | g/L | 40-55 |
| 肌酐 | μmol/L | 男53-106 / 女44-97 |
| BUN | mmol/L | 2.9-8.2 |
| eGFR | ml/min | >90 |
| 尿酸 | μmol/L | 男208-428 / 女155-357 |
| 血红蛋白 | g/L | 120-160 |
| 铁蛋白 | μg/L | 15-200 |
| 维生素D | ng/mL | >20 |
| CRP | mg/L | <10 |

### 诊断、用药、过敏、生活方式
- 主要诊断列表
- 当前用药清单（含剂量、用途、营养交互提示）
- 食物/药物过敏
- 吸烟、饮酒、运动习惯、活动水平

### 特殊人群标记
- 孕妇/哺乳期
- 儿童/青少年（<18岁）

## 输出格式

保存到 `results/<姓名>/健康档案.md`。

## 注意事项

- 不确定的数值标注"待确认"，不要猜测。
- 所有指标必须标注单位。
- 明确区分"正常"、"偏高"、"偏低"、"临界"。
- **免责声明**：本解析仅用于营养方案参考，不构成医疗诊断。
