# Nutrition Assess

基于用户的健康档案，计算个性化的每日营养目标，并生成详细的营养评估报告。

## 输入

- `results/<姓名>/健康档案.md`

## 处理步骤

### 步骤 1: 读取健康档案
提取：基本信息、异常指标、诊断疾病、用药、过敏原、营养重点关注方向。

### 步骤 2: 读取本地知识库
- `knowledge/disease-diet-rules.md`
- `knowledge/nutrient-reference.md`

### 步骤 3: 检索公开营养方案参考（强制）

必须执行至少 **3 个 WebSearch 查询**并保存到 `knowledge/references/`。

构造检索词示例：
- `"糖尿病 超重 营养治疗方案 膳食指南"`
- `"高血压 DASH饮食 中国营养方案"`
- `"慢性肾病 CKD 营养治疗 蛋白质限制"`

优先来源：中华医学会/中国营养学会指南、三甲医院营养科公开方案、丁香医生、好大夫。

保存格式：
- 文件名：`参考_营养评估_[关键词]_YYYY-MM-DD.md`
- 内容：核心要点、具体方案/数据、与本患者的关联

保存后使用 Read 工具读取文件头部 5 行验证。

### 步骤 4: 计算基础能量需求

**Mifflin-St Jeor 公式**
```
男性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 + 5
女性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 - 161
```

活动系数：
- 久坐：1.2
- 轻度活动：1.375
- 中度活动：1.55
- 高度活动：1.725

### 步骤 5: 确定体重目标

- 减重：BMI≥24 或有代谢疾病 → 每日减少 300-500 kcal
- 维持：BMI 18.5-24 且无代谢异常 → 维持 TDEE
- 增重：BMI<18.5 或营养不良 → 每日增加 200-300 kcal

特殊人群：
- 孕妇中晚期：+300-450 kcal
- 哺乳期：+500 kcal
- 儿童/青少年：查表，不使用成人公式

### 步骤 6: 疾病调整

多疾病叠加优先级（取最严格约束）：
1. 肾病
2. 痛风
3. 糖尿病
4. 高血压
5. 高脂血症
6. 肝病
7. 贫血
8. 食物过敏

### 步骤 7: 处理药物-营养交互

- 二甲双胍 → 关注维生素B12
- 利尿剂 → 注意钾
- 华法林 → 保持维生素K摄入稳定
- 他汀类 → 避免西柚/葡萄柚
- 质子泵抑制剂 → 关注钙、镁、B12吸收

### 步骤 8: 生成营养评估报告

保存到 `results/<姓名>/营养评估.md`，包含：
- 基础代谢计算（BMR、TDEE、目标能量）
- 每日营养目标（宏量营养素、微量营养素）
- 饮食原则
- 推荐/禁忌食物清单
- 药物-营养交互提示
- 三餐能量分配建议
- 参考来源
- 关键注意事项
- 免责声明

## 输出后

向用户展示营养评估摘要，确认方向正确后，提示可继续 `/exercise-plan` 或 `/meal-plan`。
