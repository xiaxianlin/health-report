# Exercise Plan

基于用户的健康档案和营养评估，生成个性化的运动处方方案。

## 输入

- `results/<姓名>/健康档案.md`
- `results/<姓名>/营养评估.md`

## 处理步骤

### 步骤 1: 读取输入文件
提取：基本信息、诊断、异常指标、用药、活动水平、能量目标、体重管理目标。

### 步骤 2: 读取本地知识库
- `knowledge/disease-diet-rules.md`
- `knowledge/nutrient-reference.md`

### 步骤 3: 检索公开运动指南参考（强制）

必须执行至少 **3 个 WebSearch 查询**并保存到 `knowledge/references/`。

检索词示例：
- `"糖尿病运动指南 ACSM 2024"`
- `"痛风患者运动指南 运动强度 乳酸"`
- `"慢性肾病 CKD 运动处方"`
- `"肥胖运动处方 有氧+力量"`

优先来源：ACSM、WHO、中华医学会运动医学分会、三甲医院康复科/运动医学科、丁香医生、好大夫。

保存格式：
- 文件名：`参考_运动处方_[关键词]_YYYY-MM-DD.md`
- 内容：核心要点、具体方案/数据、与本患者的关联

保存后使用 Read 工具验证头部 5 行。

### 步骤 4: 运动安全性评估

#### 绝对禁忌（必须告知用户不宜运动，建议咨询医生）
- 急性痛风发作期
- 空腹血糖 >16.7 mmol/L 或 <3.9 mmol/L
- 收缩压 >180 mmHg 或舒张压 >110 mmHg
- CKD 5期未透析
- 不稳定心绞痛、近期心梗
- 急性感染或发热

#### 相对禁忌与限制
| 疾病/状态 | 运动限制 |
|---|---|
| 痛风/高尿酸 | 避免高强度无氧、大量出汗 |
| CKD 3-4期 | 中低强度有氧为主 |
| 糖尿病 | 避免空腹运动，随身携带糖块 |
| 高血压 | 避免屏气动作、倒立 |
| 肥胖/超重 | 避免高冲击运动，保护关节 |

#### 药物影响
- 利尿剂 → 充分补水
- β受体阻滞剂 → 心率不可靠，改用 RPE
- 降糖药 → 餐后1-2小时运动
- 他汀类 → 关注肌肉酸痛
- 抗凝药 → 避免碰撞类运动

### 步骤 5: 运动处方设计

#### 运动类型
- 有氧：快走、游泳、骑自行车、椭圆机
- 抗阻：弹力带、自重训练、轻哑铃
- 柔韧与平衡：拉伸、八段锦、太极拳

#### 强度设定
- RPE 11-13（低至中等）为大多数慢病人群起点
- 最大心率 = 220 - 年龄
- 靶心率 = 最大心率 × 强度%

#### 频率与时长（参考 ACSM）
| 人群 | 有氧频率 | 有氧时长 | 抗阻频率 | 抗阻时长 |
|---|---|---|---|---|
| 糖尿病 | 5-7次/周 | 30-60分钟 | 2-3次/周 | 20-30分钟 |
| 痛风 | 4-5次/周 | 30-45分钟 | 2次/周 | 20分钟 |
| CKD 3-4期 | 3-5次/周 | 20-40分钟 | 2-3次/周 | 20分钟 |
| 肥胖 | 5-7次/周 | 45-60分钟 | 2-3次/周 | 20-30分钟 |

### 步骤 6: 生成运动处方报告

保存到 `results/<姓名>/运动处方.md`，包含：
- 安全性评估
- 运动处方参数（类型、强度、频率、时长、靶心率）
- 有氧/抗阻/柔韧明细
- 一周运动计划（分阶段：适应期/进阶期/巩固期）
- 单次运动流程模板
- 运动与营养协同
- 注意事项（按疾病分组）
- 循序渐进计划
- 参考来源
- 免责声明

## 输出后

向用户展示运动处方摘要，确认后继续 `/meal-plan`。
