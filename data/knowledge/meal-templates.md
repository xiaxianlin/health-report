# 配餐模板

## 一日三餐基础能量分配

| 餐次 | 能量占比 | 餐盘结构 |
|------|---------|---------|
| 早餐 | 30% | 主食 + 蛋白质 + 蔬果 |
| 午餐 | 40% | 主食 + 蛋白质 + 蔬菜(2种+) + 汤 |
| 晚餐 | 30% | 主食(减量) + 蛋白质 + 蔬菜 |

## 早餐模板

### 模板 A（中式）
- 杂粮粥/豆浆 1碗
- 水煮蛋 1个
- 蒸南瓜/红薯 1小块
- 凉拌蔬菜 1份

### 模板 B（西式）
- 全麦面包 2片
- 牛奶/酸奶 1杯(250ml)
- 鸡蛋 1个
- 水果 1份

### 模板 C（快手）
- 燕麦牛奶 1碗
- 坚果 1小把(10g)
- 水果 1份
- 水煮蛋 1个

## 午餐模板

### 模板 A（标准午餐）
- 杂粮饭 1碗(150g熟)
- 肉类/鱼类 1份(75g)
- 豆制品 1份(50g)
- 蔬菜 2份(200g)
- 汤 1碗

### 模板 B（轻食）
- 藜麦/糙米沙拉 1份
- 鸡胸肉/鱼肉 1份(75g)
- 混合蔬菜 1份(200g)
- 低脂酱汁

### 模板 C（汤面类）
- 荞麦面/全麦面 1份
- 蔬菜浇头 2份(200g)
- 肉片/蛋花 1份
- 少油少盐

## 晚餐模板

### 模板 A（标准晚餐）
- 粗粮主食 1份(100g熟)
- 清蒸鱼/白灼虾 1份(75g)
- 蔬菜 2份(200g)
- 少油烹饪

### 模板 B（清淡晚餐）
- 蔬菜汤 1碗
- 豆腐/蛋 1份
- 少量主食
- 凉拌菜 1份

### 模板 C（简易晚餐）
- 杂粮粥 1碗
- 蔬菜炒蛋 1份
- 清炒时蔬 1份

## 加餐模板（可选，150-200 kcal）

| 类型 | 示例 |
|------|------|
| 水果加餐 | 苹果1个/橙子1个/蓝莓100g |
| 坚果加餐 | 混合坚果25g |
| 奶制品加餐 | 酸奶1杯(200g) |
| 蛋白加餐 | 水煮蛋1个 |

## 一周食材多样化要求

- 每天：谷物3种+，蔬菜3种+，水果1-2种，蛋白质1-2种
- 每周：谷物5种+，蔬菜10种+，水果5种+，蛋白质4种+
- 全周至少25种不同食物

## 烹饪方式建议

| 优先推荐 | 适量使用 | 尽量避免 |
|---------|---------|---------|
| 蒸 | 炒 | 油炸 |
| 煮 | 烤 | 红烧 |
| 炖 | 煎 | 糖醋 |
| 凉拌 | 卤 | 干锅 |
| 白灼 | | 水煮鱼(重油) |
