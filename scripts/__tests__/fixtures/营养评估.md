# 营养评估报告

> 生成日期: 2025-04-07
> 基于: 健康档案（2025-09-20）

---

## 基础代谢计算

### BMR

（Mifflin-St Jeor 公式）

假设年龄35岁（IT从业人员典型年龄）：

```
BMR = 10 × 87.1 + 6.25 × 170.5 - 5 × 35 + 5
    = 871 + 1065.625 - 175 + 5
    = 1766.625 ≈ 1767 kcal/天
```

**BMR: 1767 kcal/天**

### 活动水平

- 当前活动水平: **久坐**（IT工程师，软通动力员工）
- 活动系数: **1.2**

### TDEE

```
TDEE = 1767 × 1.2 = 2120 kcal/天
```

**TDEE: 2120 kcal/天**

### 目标能量

- **肥胖 + 脂肪肝**: 目标减重，每日减少 **500 kcal**
- **调整后目标能量**: 2120 - 500 = **1620 kcal/天**
- **目标**: 3-6个月减重8-12kg（BMI从30降至约26）

## 饮食原则

### 来自脂肪肝

- **减重是首要目标**：减重3-5%可逆转脂肪肝
- 严格**戒酒**（即使非酒精性也应避免）
- 避免**果糖**，特别是含糖饮料
- 避免**加工肉类**，减少红肉
- 优先采用**地中海饮食**模式

### 来自肥胖（减重目标）

- 每日1620 kcal，制造500kcal缺口
- 每周减重0.5-1kg
- 主食以全谷物为主，至少占一半
- 控制总能量，定时定量三餐

### 来自高胆固醇

- 每日膳食胆固醇 **<300 mg**
- 限制**动物内脏**、**蛋黄**（每周3-4个）
- 减少**饱和脂肪**摄入
- 优先选择富含不饱和脂肪酸的**橄榄油**、**茶籽油**
- 增加富含钙、镁的食物

### 来自AST轻度升高（肝功能保护）

- 严格**低脂饮食**，减少肝脏负担
- 适量优质蛋白，促进肝细胞修复
- 避免肝损药物和毒素
- 增加维生素E等抗氧化营养素

## 推荐食物清单

### 主食类（全谷物优先）

- 燕麦、糙米、荞麦面、全麦面包、藜麦
- 红薯、山药、玉米（适量）

### 蛋白质类

- **鱼类**（三文鱼、鲭鱼、鲈鱼 — 每周至少2次深海鱼）
- **鸡胸肉**（去皮，优选）
- **豆制品**（豆腐、豆浆）
- **鸡蛋**（每天1个，或每周3-4个）
- **低脂牛奶/酸奶**

### 蔬菜类

- **绿叶蔬菜**（菠菜、西兰花、油麦菜）— 每天至少400g
- **十字花科蔬菜**（西兰花、花椰菜）— 有助于肝脏解毒
- **番茄、黄瓜、茄子**
- **洋葱、胡萝卜、芹菜**

### 水果类

- **低糖水果**：苹果、梨、柚子、蓝莓、草莓
- **限量**：每天200g，避免高糖水果（葡萄、荔枝、芒果）

### 脂肪来源

- **橄榄油、茶籽油**（烹饪用油，每天20-25g）
- **坚果**：核桃、杏仁（每天10-15g）

## 禁忌食物

| 食物 | 原因 | 严重程度 |
|------|------|---------|
| **酒精** | 脂肪肝 + 肝损伤风险 | **完全禁止** |
| **含糖饮料** | 果糖导致脂肪肝恶化 | **完全禁止** |
| **动物内脏** | 高胆固醇 | **完全禁止** |
| **蛋黄过量** | 胆固醇超标 | **严格限制**（每天≤1个）|
| **油炸食品** | 高脂肪、高热量 | **完全禁止** |
| **加工肉类** | 高钠、高脂、致癌风险 | **完全禁止** |
| **肥肉、猪油** | 高饱和脂肪 | **完全禁止** |
| **奶油、黄油** | 高饱和脂肪 | **严格限制** |
| **精制糖、糕点** | 高热量、低营养 | **严格限制** |

## 关键注意事项

1. **戒酒是第一优先级** — 对脂肪肝和肝功能恢复至关重要
2. **地中海饮食模式** — 优先推荐的饮食方式，富含橄榄油、鱼类、蔬果
3. **减重速度控制** — 每周0.5-1kg为宜，过快减重可能加重肝损伤
4. **胆固醇控制** — 每日<300mg，严格控制动物内脏和蛋黄
5. **定期复查** — 建议3-6个月后复查肝功能、血脂、肝脏B超
