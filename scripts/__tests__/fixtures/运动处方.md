## 六、注意事项与应急处理

### 一般注意事项

1. **运动前后监测**: 初期建议记录运动前后的心率、血压、主观感受
2. **循序渐进**: 从低强度开始，每周增加不超过10%的运动量
3. **关节保护**: 穿有缓冲功能的运动鞋，避免硬地面长时间运动
4. **血压注意**: 舒张压偏低，运动后起身时放慢速度，防止体位性低血压
5. **肝功能监测**: 每4-6周复查肝功能，AST正常后可逐步增加强度

### 疾病特殊注意

| 疾病 | 注意事项 |
|------|---------|
| **脂肪肝** | 戒酒是第一优先级；严格避免高脂饮食；坚持运动是逆转关键 |
| **肥胖** | 保护膝关节，避免跑跳；体重下降5%即可改善脂肪肝 |
| **高胆固醇** | 运动有助于改善血脂，配合低脂饮食效果更佳 |
| **AST升高** | 运动中如出现明显疲劳持续24小时以上，需降低强度 |

### 运动中不适处理

| 症状 | 处理方式 |
|------|---------|
| 头晕/眼前发黑 | 立即停止，坐下休息，补充糖水或果汁 |
| 胸闷/心悸 | 停止运动，如持续需就医 |
| 膝关节疼痛 | 停止运动，冰敷，改为游泳等无冲击运动 |
| 极度疲劳/恶心 | 降低强度或停止，休息补水 |
| 低血糖症状 | 立即停止，进食含糖食品 |

### 何时需要就医

- 运动中出现胸痛、胸闷、呼吸困难
- 关节疼痛持续不缓解
- AST升高超过正常上限2倍（>80 U/L）
- 出现黄疸、尿液深黄等肝功能恶化表现
- 体重下降过快（每周>1.5kg）
