## 饮食原则

### 来自痛风（最高优先级）
- **严格限制嘌呤**: 每日 < 150mg，避免高嘌呤食物
- **戒啤酒**: 啤酒含大量鸟嘌呤，是最危险的酒精饮料
- **限制其他酒精**: 如需饮酒，红酒少量（但减重期间不建议）
- **足量饮水**: 每天 2500-3000ml，白开水/淡茶水
- **蛋白质以蛋奶为主**: 鸡蛋（嘌呤极低~1mg/100g）、低脂牛奶
- **豆制品安全**: 豆腐、豆浆经加工后嘌呤大幅降低，可正常食用

### 来自高甘油三酯血症
- **限制精制碳水**: 白米饭、白面条减量
- **严格限制果糖**: 不喝含糖饮料、果汁、蜂蜜水
- **增加Omega-3**: 每周2次低嘌呤鱼类（鲈鱼、桂鱼）
- **控制饱和脂肪**: 减少红肉频率

### 来自超重（减重目标）
- 每日 1900 kcal，温和减重
- **应酬策略**: 点菜指南（见注意事项）

### 来自虾类过敏
- **完全禁止虾类**（基围虾、明虾、龙虾、皮皮虾等）
- 其他甲壳类（蟹）谨慎尝试
- 非甲壳类海鲜（鱼、贝类）需按嘌呤等级判断

## 推荐食物清单

### 安全蛋白质（低嘌呤）
- **鸡蛋**（每天1-2个，嘌呤极低）
- **低脂牛奶/酸奶**（≥300ml/天，促进尿酸排泄）
- **豆腐/豆浆**（加工后安全，嘌呤 ~29-55mg/100g）
- **低嘌呤鱼类**: 鲈鱼、桂鱼（≤75g/次，每周2-3次）
- **鸡胸肉**（少量，≤50g/次，煮沸去汤后嘌呤降低）

### 主食类（低嘌呤）
- 精白米/白面（低嘌呤但高GI，需搭配粗粮）
- 燕麦、红薯、山药、玉米
- 荞麦面、全麦面包

### 蔬菜类（低嘌呤，≥500g/天）
- **所有蔬菜基本安全** — 包括菠菜、蘑菇（植物嘌呤不增加痛风风险）
- 黄瓜、冬瓜（利水消肿，促进尿酸排泄）
- 芹菜、西兰花、白菜、番茄

### 水果类（低嘌呤，但注意果糖）
- 樱桃（尤其推荐，有研究显示可降低痛风发作风险）
- 苹果、梨、柚子、蓝莓
- **避免**: 荔枝、龙眼等高糖水果（果糖高）

### 其他推荐
- 咖啡（适量，黑咖啡有助降尿酸）
- 低脂奶制品（脱脂牛奶、酸奶）
- 坚果（每天一小把，10-15g）

## 禁忌食物

| 食物 | 原因 | 严重程度 |
|------|------|---------|
| **啤酒** | 高嘌呤+酒精双重打击 | **完全禁止** |
| **动物内脏**（肝、肾、脑） | 嘌呤极高（200-1000mg/100g） | **完全禁止** |
| **浓肉汤/火锅汤** | 嘌呤溶于汤中，浓度极高 | **完全禁止** |
| **沙丁鱼、凤尾鱼** | 嘌呤极高 | **完全禁止** |
| **虾类** | 过敏 + 中高嘌呤 | **完全禁止** |
| **含糖饮料/果汁** | 果糖促进尿酸生成 | **完全禁止** |
| **蜂蜜** | 高果糖 | **完全禁止** |
| 红肉（牛羊猪肉） | 中高嘌呤 | **严格限制**（≤2次/周，每次≤75g，煮沸去汤）|
| 贝类（扇贝、生蚝） | 中高嘌呤 + 过敏风险 | **严格限制** |
| 白酒/烈酒 | 酒精抑制尿酸排泄 | **严格限制** |
| 高糖水果（荔枝、龙眼） | 高果糖 | **适量控制** |
| 精制碳水（白米白面） | 高TG需要控制 | **适量控制** |

| 药物 | 注意事项 |
|------|---------|
| **非布司他** | 无直接营养交互，但需定期监测肝功能。可能引起转氨酶升高 |

## 关键注意事项

1. **戒啤酒是第一优先级** — 王先生每周3-4次啤酒是最主要的尿酸升高因素，必须完全停止
2. **应酬策略**: 点清蒸鱼/白灼鸡（不喝汤）、多点蔬菜、以茶代酒、避免火锅汤底
3. **蔬菜放心吃** — 包括菠菜和蘑菇，植物嘌呤不会增加痛风风险（2024指南明确纠正了旧观念）
4. **豆制品是安全的** — 豆腐、豆浆可正常食用，不必忌口
5. **奶制品是降尿酸利器** — 每天≥300ml低脂牛奶，是痛风患者最推荐的蛋白质来源
