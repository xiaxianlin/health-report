# 营养评估报告

> 基于健康档案生成 | 日期: 2026-04-10
> 适用对象: 张建国（53岁 男性，2型糖尿病+高血压+高脂血症）

## 基础代谢计算

### BMR

（Mifflin-St Jeor 公式）

```
BMR = 10 × 87 + 6.25 × 175 - 5 × 53 + 5
    = 870 + 1093.75 - 265 + 5
    = 1703.75 ≈ 1726 kcal/天
```

**BMR: 1726 kcal/天**

### 活动水平

- 当前活动水平: **久坐**（办公室工作）
- 活动系数: **1.2**

### TDEE

```
TDEE = 1726 × 1.2 = 2071 kcal/天
```

**TDEE: 2071 kcal/天**

### 目标能量

- 体重目标: 减重（超重+代谢疾病）
- 每日减少 **400 kcal**
- **调整后目标能量**: 2071 - 400 = **1671 ≈ 1670 kcal/天**

## 饮食原则
- 定时定量三餐，控制总热量
- 低盐饮食：每日食盐<5g
- 低脂饮食：每日烹调油<25g
- 低升糖指数食物为主：全谷物、杂豆、非淀粉蔬菜
- 每日蔬菜摄入>500g，深色蔬菜过半
- 水果选择低GI品种，每日200g以内
- 每周鱼类2次以上，优先深海鱼
- 戒烟限酒

## 禁忌食物
| 食物 | 原因 | 严重程度 |
|------|------|---------|
| 含糖饮料 | 血糖飙升 | 完全禁止 |
| 糖果糕点 | 高糖高脂 | 完全禁止 |
| 动物内脏 | 高胆固醇高嘌呤 | 严格限制 |
| 油炸食品 | 高反式脂肪酸 | 完全禁止 |
| 腌制食品 | 高钠 | 严格限制 |
| 肥肉 | 高饱和脂肪 | 严格限制 |
| 酒精 | 影响血糖+升高血脂 | 严格限制 |

## 药物-营养交互提示
- 二甲双胍 → 监测维生素B12水平，建议定期补充
- 氨氯地平 → 低盐饮食协同降压，注意钾平衡
- 阿托伐他汀 → 避免西柚/葡萄柚，监测肝功能
