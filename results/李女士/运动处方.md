### 1.3 CKD G4期运动特殊安全考量

1. **高钾血症风险**: 运动时肌肉细胞释放钾离子至血液，可致血钾急性升高。李女士血钾已处5.2 mmol/L的偏高状态，高强度运动可能诱发运动性心律失常。**必须限制为中低强度运动**，避免高强度或长时间持续运动
2. **贫血限制**: Hb 98 g/L，运动时携氧能力下降，易出现疲劳、头晕、气短。运动中需密切监测主观疲劳感
3. **血压波动**: 高血压2级患者在运动中收缩压可升高20-40 mmHg，需避免突然用力、屏气动作及等长收缩运动（如平板支撑、大重量静力训练）
4. **骨骼脆弱**: 维生素D缺乏+低血钙+高磷血症，存在肾性骨营养不良风险，需避免高冲击运动（跳跃、跑步）以防骨折
5. **营养不良**: 低白蛋白血症提示蛋白质-能量营养不良，运动消耗需控制在合理范围，避免因运动量过大进一步消耗蛋白质
6. **药物影响**: 缬沙坦（ARB）可升高血钾，运动中血钾升高的叠加效应需警惕；碳酸氢钠片提供额外钠负荷，需注意水钠平衡

## 二、运动处方（FITT-VP）

### 2.1 有氧运动处方

| 参数 | 处方 | 依据 |
|------|------|------|
| **类型（Type）** | 散步（平地）、室内踏步、固定自行车（无阻力/低阻力）、太极拳 | CKD G4期推荐低冲击运动；避免跑步、跳跃等高冲击运动 |
| **强度（Intensity）** | **低至中等强度**，Borg主观疲劳量表 11-13分（"轻松"至"稍费力"） | 高钾血症限制：避免中高强度以上运动；贫血限制携氧能力 |
| | 目标心率: **75-95 bpm**（静息心率+10-30次/分） | 间接测算法：68岁，最大心率≈152 bpm，取40-55%≈61-84 bpm，结合静息心率调整 |
| | 运动中能正常说话、不气喘 | 谈话测试作为实用监测指标 |
| **频率（Frequency）** | **每周5-7天**，尽可能每天 | CKD指南推荐"大部分天数"，习惯性活动优于间歇性 |
| **时长（Time）** | **每次20-30分钟**，累计每周150-180分钟 | 初期从15分钟开始，逐步增至30分钟；避免单次超过45分钟 |
| **总量（Volume）** | 每周150-180分钟低至中等强度有氧运动 | 低于普通成人推荐量（150-300分钟中等强度），因CKD G4+高钾+贫血需保守 |
| **进阶（Progression）** | 每2-4周增加5分钟/次，或每周增加1天 | 缓慢进阶，密切观察耐受性 |

### 2.2 抗阻运动处方

| 参数 | 处方 | 依据 |
|------|------|------|
| **类型** | 自身体重轻度抗阻（坐姿起立、扶墙俯卧撑、弹力带轻度阻力）、坐姿哑铃（0.5-1kg） | CKD G4避免大重量训练；低重量高重复有助于维持肌力而不增加肾脏负担 |
| **强度** | Borg 11-13分，感觉很轻松到稍费力 | 避免屏气（Valsalva动作），防止血压骤升和血钾波动 |
| **频率** | 每周2-3次，非连续日 | 隔天进行，给予恢复时间 |
| **时长** | 每次15-20分钟，每个动作2组×10-12次 | 低容量抗阻，防止过度疲劳 |
| **注意事项** | 呼吸节奏均匀（发力时呼气，放松时吸气）；禁止屏气；禁止等长收缩 | CKD+高血压患者等长收缩可致血压骤升 |

### 2.3 柔韧性与平衡训练

| 参数 | 处方 | 依据 |
|------|------|------|
| **类型** | 坐姿拉伸、站立拉伸（扶支撑物）、太极拳/八段锦、坐姿瑜伽 | 老年人防跌倒；CKD患者柔韧性训练安全可行 |
| **强度** | 轻柔拉伸，无痛范围 | 维生素D缺乏+低血钙有骨折风险，拉伸需温和 |
| **频率** | 每天或至少每周5次 | 灵活性训练可每天进行 |
| **时长** | 每次10-15分钟 | 可融入有氧运动的热身/整理活动中 |

### 2.4 禁止/限制的运动类型

| 运动类型 | 限制级别 | 原因 |
|----------|----------|------|
| 高强度有氧（快跑、HIIT、竞技运动） | **完全禁止** | 血钾急性升高风险、肾脏血流急剧减少 |
| 高冲击运动（跳绳、篮球、排球） | **完全禁止** | 肾性骨营养不良，骨折风险 |
| 等长收缩运动（平板支撑、靠墙静蹲>30秒） | **完全禁止** | 血压骤升风险 |
| 大重量力量训练（>3kg哑铃/器械） | **完全禁止** | 屏气致血压骤升、横纹肌溶解风险 |
| 高温运动（热瑜伽、桑拿后运动） | **完全禁止** | 脱水加重肾损伤、电解质紊乱 |
| 长时间耐力运动（>45分钟连续有氧） | **严格限制** | 蛋白质消耗增加、血钾持续升高 |
| 登山/高原运动 | **严格限制** | 缺氧加重贫血症状 |
| 游泳 | **需评估** | 若水温合适、水质清洁可以考虑，但需注意水分摄入和感染风险 |

## 四、单次运动流程

### 4.1 标准运动流程（以散步为例）

```
总时长约 40-50 分钟
│
├── 运动前准备（5-10分钟）
│   ├── 测量血压（如条件允许）
│   ├── 确认当日感觉良好：无头晕、胸闷、心悸、明显乏力
│   ├── 穿舒适运动鞋和宽松衣物
│   ├── 携带手机（紧急联系用）
│   ├── 饮水100-150ml（小口慢饮）
│   └── 告知家人运动计划
│
├── 热身阶段（5-8分钟）
│   ├── 原地慢步2分钟
│   ├── 颈部缓慢旋转（左右各5次）
│   ├── 肩部环绕（前后各10次）
│   ├── 手腕脚踝环绕各10次
│   ├── 站立小腿拉伸（扶墙，每侧15秒）
│   └── 大腿前侧拉伸（扶椅背，每侧15秒）
│
├── 主体运动（20-30分钟）
│   ├── 散步（平地）或固定自行车或太极拳
│   ├── 速度：能正常说话的节奏
│   ├── 每10分钟评估一次主观感受
│   ├── 运动中小口饮水（总量不超过200ml）
│   └── 如有任何不适，立即停止
│
├── 整理活动（5-8分钟）
│   ├── 慢步2-3分钟（逐渐降低心率）
│   ├── 站立拉伸（全身主要肌群，每个动作保持15-20秒）
│   │   ├── 小腿拉伸
│   │   ├── 大腿前后侧拉伸
│   │   ├── 肩臂拉伸
│   │   └── 背部轻柔旋转
│   └── 深呼吸3-5次（鼻吸口呼）
│
└── 运动后（5分钟）
    ├── 休息坐下，饮水100-150ml
    ├── 记录运动内容、时长、主观感受
    └── 如条件允许，再次测量血压
```

### 4.2 运动中停止运动指征（红色警报）

出现以下任一症状时，**必须立即停止运动并休息**：

- 胸闷、胸痛或压迫感
- 心悸、心跳不规律
- 头晕、眼前发黑
- 恶心、呕吐
- 异常气短（无法正常说话）
- 肌肉无力、发麻
- 大量出汗或面色苍白
- 头痛加重
- 任何关节或骨骼疼痛

**停止后处理**: 就地坐下或半卧位休息，饮水，如症状5分钟内不缓解或加重，立即拨打120急救电话。

## 五、运动与营养协同

### 5.1 运动时间与饮食安排

| 时间节点 | 营养建议 | 原因 |
|----------|----------|------|
| **运动前1-2小时** | 可少量进食（如半碗白粥或1片白面包）| 预防低血糖；避免空腹运动以防贫血相关的头晕 |
| **运动前30分钟** | 饮水100-150ml | 保证水合状态，但不过量（CKD需控水） |
| **运动中** | 小口饮水，总量100-200ml | 防止脱水加重肾损伤 |
| **运动后30分钟内** | 不需额外加餐；按正常饮食时间进餐 | CKD低蛋白饮食下不宜额外补充蛋白；热量已由正餐保证 |
| **运动后** | 补铁药（琥珀酸亚铁）可在运动后餐后服用 | 运动后血液循环加速，配合维C促进铁吸收 |

### 5.2 特殊协同注意

1. **禁止运动补剂**: 蛋白粉、肌酸、电解质饮料、运动能量棒等**全部禁止**。CKD低蛋白饮食需严格33g/天，任何补剂都可能超出限制
2. **禁止运动饮料**: 市售运动饮料含钾、磷、糖均较高，不适合CKD G4期+高钾+高磷患者。运动中补水只需白开水
3. **热量保障**: 运动消耗约100-150 kcal/次（低强度30分钟），需在每日1500 kcal总量中预留，主要通过增加碳水（如藕粉、粉丝）补足，而非增加蛋白质
4. **司维拉姆随餐服用**: 如果运动时间在餐后附近，确保司维拉姆已随餐服用，不因运动改变服药时间
5. **避免运动后暴饮暴食**: 运动后可能食欲增加，但仍需遵守33g蛋白质限制，可用藕粉冲泡或白糖水补充热量

### 5.3 运动与血糖管理

- 糖尿病前期患者运动有助于改善胰岛素敏感性
- 避免空腹运动，预防低血糖（虽然目前仅为糖尿病前期，风险较低）
- 随身可携带1-2块糖果，以防运动中低血糖（出现手抖、心慌、出冷汗时含一颗）
- 运动后不以高糖食物"奖励"自己

## 六、注意事项

### 6.1 CKD G4期专属注意事项

1. **血钾监测**: 运动开始前和开始后的第1个月，建议每周检测一次血钾；稳定后每月检测一次。如血钾>5.5 mmol/L，暂停运动计划并就医
2. **避免脱水**: CKD患者脱水可导致肾功能急剧恶化。运动中需少量多次饮水，避免大量出汗（高温天气避免户外运动）
3. **避免横纹肌溶解**: CKD患者严禁高强度运动，横纹肌溶解释放肌红蛋白可堵塞肾小管，造成急性肾损伤
4. **肾功能监测**: 每3个月复查eGFR、肌酐，如eGFR下降>5 mL/min或肌酐持续上升，需调整运动计划
5. **蛋白尿观察**: 如运动后尿液泡沫明显增多，需告知肾内科医生

### 6.2 高血压专属注意事项

1. **运动前血压检查**: 如运动前血压>160/100 mmHg，当日暂停运动
2. **避免等长收缩**: 屏气用力可致收缩压瞬间升高至200 mmHg以上，极其危险
3. **避免突然用力**: 运动开始和结束要缓慢过渡，避免突然加速或减速
4. **寒冷天气注意**: 低温可致血管收缩、血压升高，冬季户外运动需保暖

### 6.3 贫血专属注意事项

1. **疲劳感监测**: 如日常活动中频繁感到明显乏力、气短，应暂停运动增加休息
2. **血红蛋白目标**: Hb<90 g/L时仅做室内轻柔活动（坐姿拉伸），暂缓有氧运动计划
3. **起立缓慢**: 运动后或久坐后起立需缓慢，防止体位性低血压（贫血+降压药的叠加效应）

### 6.4 老年人专属注意事项

1. **防跌倒**: 所有站立运动需靠近墙壁或坚固家具；户外散步需平坦路面；穿防滑鞋
2. **平衡训练**: 太极拳和八段锦是最佳选择，兼具平衡训练和柔韧性训练
3. **骨密度保护**: 维生素D缺乏+低血钙增加骨折风险，避免扭转和冲击动作
4. **陪伴运动**: 建议与家人或朋友一起运动，或携带手机保持通讯

### 6.5 环境与装备注意事项

1. **温度**: 避免在>30度或<5度的环境中户外运动
2. **时间**: 建议上午9-10点或下午4-5点运动，避免清晨（血压晨峰）和晚间（影响睡眠）
3. **鞋履**: 穿有缓冲的平底运动鞋，避免穿拖鞋或皮鞋运动
4. **地面**: 选择平坦、不湿滑的路面或场地

| 指标 | 目标 |
|------|------|

**维持期监测要求**:
- 每月复查血钾、血磷、血红蛋白、白蛋白
- 每3个月复查eGFR、血压全面评估
- 每6个月评估运动计划，根据病情调整
- 出现任何病情变化，回到阶段1重新评估

### 降级标准

出现以下情况时，运动计划需降级或暂停:

| 情况 | 处理 |
|------|------|
| 血钾>5.5 mmol/L | 暂停所有运动，就医调整后重新评估 |
| Hb<90 g/L | 暂停有氧运动，仅保留坐姿柔韧性训练 |
| 血压>160/100 mmHg | 暂停运动，调整降压方案 |
| eGFR下降>5 mL/min（3个月内）| 暂停运动，肾内科评估 |
| 运动后蛋白尿明显增加 | 暂停运动，肾内科评估 |
| 出现心悸、胸闷等不适 | 立即暂停，就医评估 |
| 感冒、发热、腹泻期间 | 暂停运动至完全恢复 |
| 任何急性疾病 | 暂停运动，痊愈后从阶段1重新开始 |
