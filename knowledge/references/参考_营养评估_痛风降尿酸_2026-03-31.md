# 痛风/高尿酸血症饮食治疗方案

> 基于权威来源整理，主要包括：国家卫生健康委《成人高尿酸血症与痛风食养指南（2024年版）》、Mayo Clinic痛风饮食指南、CCHRC降尿酸饮食建议、航天中心医院、金华市卫健委科普等。

---

## 一、疾病背景与诊断标准

- **高尿酸血症诊断标准**：正常膳食状态下，非同日2次检测空腹血尿酸水平 > 420 umol/L
- **我国成人患病率**：高尿酸血症 14%，痛风 0.86%~2.20%，男性高于女性
- **尿酸来源**：约 80% 为内源性（人体自身细胞代谢产生），约 20% 为外源性（食物摄入）
- **痛风分期**：
  1. 无症状高尿酸血症期
  2. 急性痛风性关节炎期（关节红肿热痛，疼痛剧烈）
  3. 痛风间歇期（两次急性发作之间）
  4. 慢性痛风性关节炎期（持续疼痛，可伴痛风石）

## 二、国家卫健委2024版指南 -- 七项饮食原则

| 序号 | 原则 | 核心内容 |
|------|------|---------|
| 1 | 食物多样，限制嘌呤 | 保持膳食多样性，重点限制高嘌呤食物 |
| 2 | 蔬奶充足，限制果糖 | 蔬菜每日 >= 500g，奶制品 >= 300mL |
| 3 | 足量饮水，限制饮酒 | 每日饮水 2000~3000mL |
| 4 | 科学烹饪，少食生冷 | 少油炸煎烤，提倡汆煮；盐 <= 5g/天，油 25~30g/天 |
| 5 | 吃动平衡，健康体重 | 每周减重 0.5~1.0kg 为宜 |
| 6 | 辨证辨体，因人施膳 | 中医食养个性化调理 |
| 7 | 因地因时，分期调理 | 根据病情阶段调整饮食 |

## 三、嘌呤含量食物分类（2024版指南标准）

### 分类定义

| 分类 | 嘌呤含量 (mg/100g) | 食用建议 |
|------|---------------------|---------|
| **高嘌呤食物** | 150~1000 mg/100g | **避免食用** |
| **较高嘌呤食物** | 75~150 mg/100g | 限制食用 |
| **较低嘌呤食物** | 30~75 mg/100g | 可适量食用 |
| **低嘌呤食物** | < 30 mg/100g | 放心食用 |

### 高嘌呤食物（必须避免，> 150 mg/100g）

| 类别 | 具体食物 |
|------|---------|
| **动物内脏** | 肝、肾、脑、心、胰脏 |
| **特定海鲜** | 沙丁鱼、凤尾鱼（鲥鱼）、青花鱼（马鲛鱼）、鲱鱼、带壳海鲜（虾、蟹、贝类） |
| **肉汤类** | 浓肉汤、肉汁、火锅汤、骨头汤、高汤 |
| **禽类** | 鹅、火鸡 |
| **干豆类** | 红豆、绿豆、黄豆、眉豆、扁豆（干豆状态） |
| **其他** | 鱼卵、发酵鱼、鱼露、虾酱、汤块 |

### 较高/中等嘌呤食物（限制食用，75~150 mg/100g）

| 类别 | 具体食物 |
|------|---------|
| **红肉** | 猪肉、牛肉、羊肉 |
| **部分鱼类** | 鳕鱼、鳗鱼、大比目鱼、鲈鱼、三文鱼 |
| **部分蔬菜** | 芦笋、西兰花、椰菜花、菠菜、蘑菇、青豌豆 |
| **谷类** | 全麦面包、燕麦片、糙米、小麦胚芽 |
| **豆制品** | 纳豆（110 mg/100g） |

### 低嘌呤食物（放心食用，< 50 mg/100g）

| 类别 | 具体食物 | 嘌呤含量参考 |
|------|---------|-------------|
| **蛋类** | 鸡蛋、鹅蛋 | ~1 mg/100g |
| **奶制品** | 低脂/脱脂牛奶、酸奶、奶酪 | 极低 |
| **精制谷物** | 白米、白面包、面条、馒头 | 低 |
| **薯类** | 土豆、芋头、红薯 | 5~13 mg/100g |
| **低嘌呤粗粮** | 玉米面(12)、地瓜(13)、高粱米(15)、土豆淀粉(5) | < 15 mg/100g |
| **多数蔬菜** | 白菜、黄瓜、冬瓜、南瓜、苦瓜、丝瓜、茄子、芹菜、白萝卜、番茄 | 低 |
| **水果** | 苹果、橙子、香蕉、樱桃、草莓、梨 | 低 |
| **豆制品(加工后)** | 豆浆(29 mg/100g)、豆腐 | 加工后嘌呤显著降低 |
| **其他** | 海参、海蜇、花生酱、坚果 | 低 |

## 四、核心饮食策略：四吃四不吃

### 四吃（推荐）

#### 1. 水果蔬菜
- 每日新鲜蔬菜 >= 500g，深色蔬菜（紫甘蓝、胡萝卜）占一半以上
- 推荐水果：樱桃（有降尿酸证据）、草莓（富含维生素C）、苹果、橙子
- 植物性嘌呤人体利用率低，多数蔬菜可放心食用

#### 2. 奶制品
- 每日 >= 300mL 低脂或脱脂牛奶或等量奶制品
- 乳蛋白为优质蛋白，可促进尿酸排泄

#### 3. 足量饮水
- 每日饮水 2000~3000mL
- 优先白开水，可选柠檬水、淡茶、无糖咖啡、苏打水
- 避免浓茶、浓咖啡、生冷饮品
- 促进尿酸排泄，改善尿酸结晶沉积

#### 4. 吃动平衡
- 保持健康体重
- 每周减重 0.5~1.0 kg，避免过度节食或减重过快
- 规律运动

### 四不吃（避免/限制）

#### 1. 限制果糖
- 果糖诱发代谢异常和胰岛素抵抗，升高尿酸
- 避免：含糖饮料、鲜榨果汁、果葡糖浆、果脯蜜饯
- 研究证实含糖饮料增加痛风发生率，且呈剂量效应

#### 2. 限制饮酒
- 酒精促使尿酸生成增加，同时乳酸升高抑制尿酸排泄
- **黄酒**：嘌呤含量最高
- **啤酒**：嘌呤含量次高，且含酒精双重打击
- **白酒**：嘌呤低但酒精度高，乳酸堆积抑制尿酸排泄
- **红酒**：也应控制摄入量
- **预调酒**：含酒精不可忽略，注意辨别
- 痛风急性发作期严格禁酒

#### 3. 少食生冷
- 避免冰激凌、生冷海鲜
- 生冷食物损伤脾胃，可导致尿酸盐结晶析出增加
- 每日食盐 <= 5g，烹调油 25~30g

#### 4. 限制嘌呤
- 避免动物内脏（肝、肾、心等）
- 腊制、腌制、熏制肉类嘌呤和盐分均高，不宜食用
- 肉类推荐汆煮后食用（嘌呤溶于水），尽量不喝汤

## 五、常见认知纠正

### 误区1："啤酒+海鲜"是食物相克
**事实**：并非两者相互作用，而是两者各自嘌呤含量都高，叠加后总嘌呤摄入量过高。单独大量食用任一种也可能诱发痛风。"啤酒中维生素B1促进嘌呤转化为尿酸"的说法已被证伪。

### 误区2：豆制品不能吃
**事实**：干豆嘌呤高（大豆 218 mg/100g），但加工为豆腐、豆浆后嘌呤大幅降低（豆浆仅 29 mg/100g）。豆制品低脂低胆固醇、含优质蛋白，对痛风患者有益。

### 误区3：高嘌呤蔬菜不能吃
**事实**：研究未显示芦笋、菠菜等高嘌呤蔬菜增加痛风风险。植物性嘌呤人体利用率低，蔬菜可放心食用。

### 误区4：只控制嘌呤就够了
**事实**：果糖、酒精、高脂饮食同样重要。过度限制动物性食物而主食过多，可能引起胰岛素抵抗，反而影响尿酸排出。均衡饮食才是关键。

### 误区5：鸡蛋和牛奶不能吃
**事实**：鸡蛋（蛋白嘌呤极低，约 1 mg/100g）和低脂牛奶是痛风患者的理想优质蛋白来源，牛奶还能促进尿酸排泄。

## 八、推荐食材与禁忌食材速查

### 放心食用（绿色清单）
- **主食**：大米、小米、白面包、面条、馒头、玉米面、红薯、土豆
- **蛋白**：鸡蛋、低脂牛奶、脱脂酸奶、豆腐、豆浆、海参、海蜇
- **蔬菜**：白菜、黄瓜、冬瓜、南瓜、苦瓜、丝瓜、茄子、芹菜、白萝卜、番茄、生菜
- **水果**：樱桃、草莓、苹果、橙子、梨、香蕉
- **饮品**：白开水、柠檬水、淡茶、无糖咖啡、苏打水

### 限制食用（黄色清单，少量/偶尔）
- **肉类**：瘦猪肉、瘦牛肉、瘦羊肉（每日总量 <= 150g）
- **鱼类**：鲈鱼、鳕鱼、三文鱼（适量，每次 <= 75g）
- **蔬菜**：菠菜、芦笋、蘑菇、花椰菜（可食用，不需过度担心）
- **粗粮**：燕麦、全麦面包、糙米（粗细搭配即可）
- **酒精**：红酒（少量，急性期禁酒）

### 严格避免（红色清单）
- **内脏**：肝、肾、脑、心、胰脏
- **海鲜**：沙丁鱼、凤尾鱼、贝类、虾、蟹、鱼卵
- **汤类**：浓肉汤、火锅汤、骨头汤、肉汁
- **酒类**：啤酒、黄酒、高度白酒
- **饮品**：含糖饮料、果葡糖浆饮料、鲜榨果汁（大量）
- **加工食品**：腊肉、腌肉、熏肉、香肠
- **其他**：鹅肉、火鸡、油炸食品

## 九、啤酒与海鲜 -- 详细分析

### 啤酒对尿酸的三重打击
1. **嘌呤含量**：啤酒本身含有较高嘌呤（在酒类中排第二，仅次于黄酒）
2. **酒精效应**：酒精代谢产生乳酸，乳酸与尿酸竞争排泄通道，导致尿酸潴留
3. **促进代谢**：酒精加速体内嘌呤分解代谢，使血尿酸急速升高

### 海鲜风险等级
| 风险等级 | 海鲜类型 |
|---------|---------|
| **极高（避免）** | 沙丁鱼、凤尾鱼、鲭鱼、鲱鱼、鱼卵、贝类 |
| **较高（少食）** | 虾、蟹、鱿鱼、三文鱼、大比目鱼 |
| **低（可食）** | 海参、海蜇 |

### 关键结论
- "啤酒+海鲜"不是食物相克，而是嘌呤叠加效应
- 任何高嘌呤食物 + 任何酒精饮料 = 高风险组合
- 健康人群偶尔食用不必过度恐慌，但痛风患者必须严格避免
- 动物内脏的嘌呤含量甚至高于多数海鲜，是最需避免的食物

## 十、不同分期的饮食调整

### 急性痛风发作期
- 严格限制嘌呤，每日嘌呤摄入 < 150 mg
- 以低嘌呤食物为主：鸡蛋、牛奶、精制谷物、多数蔬菜水果
- 大量饮水 >= 3000mL/天
- 绝对禁酒
- 及时就医，不可仅靠饮食控制

### 痛风间歇期
- 可适度放宽，但仍以低嘌呤为主
- 肉类总量 <= 150g/天，可搭配畜禽肉和海鲜各 <= 75g
- 避免高嘌呤食物和酒精
- 保持健康体重

### 无症状高尿酸血症期
- 均衡饮食，避免暴饮暴食
- 控制果糖和酒精摄入
- 多饮水，多吃蔬菜水果
- 保持运动和健康体重
