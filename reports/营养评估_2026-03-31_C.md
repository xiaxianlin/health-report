## 一、基础代谢计算

### BMR（Mifflin-St Jeor 公式 — 女性）
```
BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄(岁) - 161
    = 10 × 55 + 6.25 × 158 - 5 × 68 - 161
    = 550 + 987.5 - 340 - 161
    = 1036.5 kcal/天 ≈ 1037 kcal/天
```

### 活动水平
- 当前活动水平: **轻度活动**（退休教师，每日散步30分钟）
- 活动系数: **1.375**

### TDEE（总能量消耗）
```
TDEE = 1037 × 1.375 = 1426 kcal/天
```

### CKD热量需求调整
- CKD G4期低蛋白饮食需要充足热量防止营养不良: 25-35 kcal/kg/天
- 范围: 55 × 25 = 1375 kcal 至 55 × 35 = 1925 kcal
- TDEE为1426 kcal，在此范围内但偏低
- 考虑到低白蛋白血症（33 g/L）和贫血需额外营养支持，同时避免加重肾脏负担
- **最终目标能量: 1500 kcal/天**
  - 取TDEE附近，略高于TDEE以防止蛋白质-能量营养不良（PEW）
  - 约合 27.3 kcal/kg，在推荐范围内

## 三、疾病冲突分析与处理策略

### 冲突1: CKD蛋白质限制 vs 贫血/低白蛋白

| 方面 | CKD要求 | 贫血/低白蛋白要求 | 处理策略 |
|------|---------|-----------------|---------|
| 蛋白质 | 限制33g/天 | 需要充足蛋白 | 严格33g上限，但≥50%高生物价蛋白 |
| 热量 | 防止PEW | 需要营养支持 | 保证1500kcal，用碳水/脂肪补足热量缺口 |
| 铁 | — | 需要heme铁 | 药物补铁为主（琥珀酸亚铁），膳食为辅 |

**核心策略**: 低蛋白+高热量。蛋白质全部分配给高生物价蛋白（蛋清>鱼肉>鸡胸肉），用植物油和淀粉类主食填满热量缺口。铁的补充依赖药物，膳食不过度追求含铁食物（红肉/肝脏磷和蛋白都高）。

### 冲突2: 限钾 vs DASH降压

| 方面 | 高血压(DASH) | CKD高钾血症 | 处理策略 |
|------|-------------|------------|---------|
| 钾 | 推荐3500-4700mg | 限制<2000mg | **CKD优先**，限钾为第一原则 |
| 降压策略 | 富钾食物 | 限钾 | 改用限钠策略降压，<2000mg钠/天 |
| 蔬果 | 大量 | 需选择低钾品种 | 选低钾蔬菜，焯水去钾后食用 |

**核心策略**: 放弃DASH高钾推荐，以限钠作为主要降压饮食手段。蔬菜全部焯水/浸泡处理降低钾含量。

### 冲突3: 限磷 vs 糖尿病前期碳水选择

| 方面 | 糖尿病前期 | CKD限磷 | 处理策略 |
|------|-----------|---------|---------|
| 主食 | 推荐全谷物/粗粮 | 全谷物含磷高 | **限磷优先**，选择精制低磷主食 |
| 低GI | 粗粮低GI | — | 用低磷低GI食物替代（如粉丝、山药） |

**核心策略**: 放弃全谷物，改用精制低磷主食（白米、面条、粉丝），搭配低磷低GI食物（山药、藕粉），用蔬菜纤维替代粗粮纤维。

### 冲突4: 补钙 vs 高草酸

| 方面 | 低血钙需要补钙 | CKD注意事项 | 处理策略 |
|------|-------------|------------|---------|
| 钙源 | 奶制品含钙高 | 奶制品含磷也高 | 限制奶制品，选择低磷钙源 |
| 草酸 | — | 高草酸影响钙吸收 | 避免菠菜等高草酸与钙同食 |

**核心策略**: 奶制品因含磷高仅限少量（≤100ml/天），钙以骨化三醇辅助吸收为主，膳食选择低磷钙源（如小白菜焯水后含钙高磷低）。

## 五、禁忌/限制食物清单

### 完全禁止

| 食物 | 原因 |
|------|------|
| **坚果类**（核桃、花生、瓜子、杏仁等）| 磷极高（>500mg/100g），钾也高 |
| **全谷物**（糙米、燕麦、全麦、荞麦）| 磷偏高，与CKD限磷冲突 |
| **豆类/豆制品大量**（黄豆、黑豆、大量豆腐）| 磷偏高，少量豆腐可接受 |
| **香蕉** | 钾256mg/100g，高钾血症禁止 |
| **橙子/橘子/柚子大量** | 钾>150mg/100g，少量可接受 |
| **土豆/红薯大量** | 钾>300mg/100g，焯水少量可 |
| **番茄大量** | 钾偏高，少量做调味可 |
| **菠菜生食** | 高钾(311mg)+高草酸，影响钙吸收 |
| **加工肉类**（火腿、香肠、培根）| 高磷（含磷酸盐添加剂）+高钠 |
| **可乐/碳酸饮料** | 含磷酸添加剂 |
| **奶制品大量**（>200ml/天）| 磷偏高（牛奶100mg/100ml） |
| **动物内脏** | 磷300-500mg/100g |
| **浓肉汤/骨头汤** | 大量磷和嘌呤溶出 |
| **腌制食品** | 高钠，加重高血压 |

## 六、药物-营养交互提示

| 药物 | 注意事项 |
|------|---------|
| **缬沙坦 160mg qd** | ARB类药物可升高血钾。当前血钾5.2已偏高，必须严格限钾饮食。避免使用低钠盐（含氯化钾）。定期监测血钾 |
| **氨氯地平 5mg qd** | 无显著营养交互。钙通道阻滞剂，不影响主要营养素吸收 |
| **碳酸氢钠片 0.5g tid** | 提供额外钠负荷（每片含钠约137mg，三片=411mg钠），需计入每日钠总量中。与其他限钠措施配合 |
| **骨化三醇 0.25μg qd** | 活性维生素D，促进肠道钙磷吸收。注意：虽然促进钙吸收有益于低血钙，但也促进磷吸收，需与司维拉姆配合使用。避免额外补充大剂量钙剂 |
| **司维拉姆 0.8g tid** | 磷结合剂，**必须随餐服用**才能发挥降磷效果。可能减少脂溶性维生素（A、D、E、K）吸收，但不影响骨化三醇（已为活性形式）。注意便秘副作用，保证膳食纤维摄入 |
| **琥珀酸亚铁 0.1g tid** | **餐后服用**以减少胃肠刺激。避免与茶、咖啡同服（鞣酸抑制铁吸收）。可搭配维C促进吸收。与司维拉姆无直接冲突，但建议间隔服用 |

## 七、蔬菜降钾处理方法（CKD患者必备技能）

由于李女士血钾5.2偏高，所有蔬菜必须经过降钾处理：

| 方法 | 操作 | 降钾效果 |
|------|------|---------|
| **焯水法（推荐）** | 切小块 → 沸水煮2-3分钟 → 捞出弃汤 → 再烹饪 | 可降低钾40-60% |
| **浸泡法** | 切薄片 → 冷水浸泡2小时以上 → 换水2-3次 | 可降低钾20-40% |
| **弃汤法** | 所有蔬菜煮汤后只吃菜不喝汤 | 钾大量溶于水中 |
| **避免生食** | 所有蔬菜烹饪后食用，避免凉拌生吃 | 减少钾摄入 |

**铁律**: 蔬菜汤/菜汤不要喝，钾全在汤里！

## 九、水分管理

- CKD G4期尿量可能减少，需根据实际尿量调整
- 基础建议: **1500-2000 ml/天**（含食物中水分）
- 若尿量正常（>1500ml/天）: 饮水1500ml左右
- 若尿量减少（<1000ml/天）: 饮水=前一日尿量+500ml
- 注意: 碳酸氢钠片需用水送服，计入总量
- 避免一次性大量饮水，分次少量

## 十一、关键注意事项

1. **蛋白质精确控制是生命线** — 每天严格33g，不可随意增加肉类摄入，哪怕感觉"营养不够"，应通过增加碳水和油脂来补足热量
2. **限钾比降压更重要** — 血钾5.2已超上限，高钾血症可致心律失常甚至猝死。所有蔬菜必须焯水，禁止喝菜汤，禁止低钠盐（含钾）
3. **热量充足才能保住白蛋白** — 低白蛋白血症(33g/L)的根源是热量不足导致蛋白质被燃烧供能。即使食欲不佳也要保证1500kcal
4. **司维拉姆必须随餐吃** — 不随餐吃等于白吃，它是通过与食物中的磷结合来降磷的
5. **补铁药餐后吃，配维C** — 琥珀酸亚铁餐后服用减少胃肠刺激，搭配维C（如几片苹果）促进吸收，但不喝橙汁（高钾）
6. **不用低钠盐** — 低钠盐用氯化钾替代氯化钠，对高钾血症患者极其危险
7. **定期监测** — 建议每月复查血钾、血磷、血钙、血红蛋白、白蛋白，3个月复查eGFR
